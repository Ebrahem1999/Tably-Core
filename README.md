# Tably Core 

A reusable React component library for restaurant QR menu applications.

## Installation

```bash
npm install @tably/core
```

## Usage

```javascript
import { CartButton, useCart, MenuPage } from '@tably/core'
import '@tably/core/styles'
```

## Components

- CartButton
- CartDrawer  
- CategoryTabs
- ImageLightbox
- ItemCard
- LanguageSwitcher
- OptionModal
- ReservationModal
- SearchBar

## Pages

- MenuPage
- ContactPage
- QRPage
- ReservationPage
- TermsPage
- PrivacyPage
- AccessibilityPage

## Contexts

- AnimationProvider/useAnimation
- CartProvider/useCart

## Building

```bash
npm run build:lib
```
